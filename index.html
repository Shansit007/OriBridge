How to use in PageMaker

  1. Type / paste your Unicode Odia (from MS Word, Nirmala UI, web, etc.) into the left box.
  2. Click Convert, then Copy Output.
  3. In Adobe PageMaker 7.0, paste the copied text into a text frame.
  4. Select the pasted text and apply the AkrutiOriSarala-99 font.

100% offline · no login · no backend · your text never leaves this page.
